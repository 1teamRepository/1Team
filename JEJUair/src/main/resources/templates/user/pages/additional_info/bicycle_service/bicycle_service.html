캐링백이 준비되어 있어요
자전거 캐링백 서비스란?
자전거 여행을 떠나는 고객님의 편의를 위해, 자전거 위탁 시 필요한 캐링백을 대여해드리는 예약 서비스입니다.
캐링백 이용순서
  • 1. 캐링백 수령
  • 2. 자전거 패킹
  • 3. 위탁수하물 처리
    (보상/파손 위약서 작성)
  • 4. 출발카운터 체크리스트 확인 후 반납
  • 출/도착 시 사용자 본인이 직접 분해 및 조립을 직접 진행하여야 하며, 분해 조립으로 발생한 파손은 책임지지 않습니다.
  • 분해 및 조립에 필요한 공구는 제공되지 않습니다. (공구 반드시 지참)
  • 전자식 구동계 160WH 또는 8G 초과는 휴대 및 위탁수하물 처리가 되지 않습니다.
  • 전자식 구동계 160WH 미만인 경우 전원을 off하거나 단선하여야 합니다.
    • 프레임 내장형 (휴대: 단선이 안되는 경우 배터리 분리 후 기내반입, 위탁수하물: 단선)
    • 스램 제품 (휴대: 배터리 탈착 후 기내반입, 위탁수하물: 불가)
캐링백 사양
분해조립 방법
이렇게 이용하세요
  • STEP 01
    국내선 예약할 때, 부가서비스를 선택하세요
    (이미 항공권 예약을 하셨다면 마이페이지에서 구매 가능합니다.)
  • STEP 02
    수하물 구매 화면에서 하단에 있는 스포츠용품 항목을 찾아보세요.
  • STEP 03
    ‘위탁 수수료+자전거 캐링백 대여’ 항목을 선택하시면 캐링백 예약 끝!
  • 서비스 가능 노선 : 김포-제주 노선
    • 김포발: 103 / 107 / 135 / 117 / 119 / 155 / 125 / 137 편
    • 제주발: 106 / 142 / 114 / 116 / 144 / 138 / 146/ 126 편
  • 캐링백 서비스가 마감되었을 경우 해당 상품이 보이지 않습니다.
  • 캐링백 서비스에는 스포츠용품 위탁 수수료가 포함되어 있습니다.
